# Character Checker - Educational Character Analysis Tool

## Overview
Character Checker is an educational Java application designed to help users understand character encoding, classification, and string manipulation in Java. The application provides various tools for analyzing characters and strings, making it a useful learning resource for Java beginners.

## Features

### 1. Single Character Analysis
- Categorizes characters (letter, digit, whitespace, etc.)
- Shows Unicode and ASCII values
- Displays binary and hexadecimal representations
- Provides detailed character properties

### 2. String Analysis
- Character type statistics and percentages
- Case distribution (uppercase vs lowercase)
- String reversal demonstration

### 3. Text File Processing
- Word count estimation
- Character frequency analysis
- Character type distribution

### 4. Character Encoding Tutorial
- Explanation of ASCII vs Unicode
- Overview of character categories in Java
- Information about Unicode character ranges
- Interesting facts about character encoding

## Requirements
- Java 17 or higher
- Command line interface

## Usage
1. Compile the Java files:
   ```
   javac *.java
   ```

2. Run the application:
   ```
   java CharacterChecker
   ```

3. Follow the menu prompts to use different features

## Educational Value
This application is designed as a learning tool to demonstrate:
- Character classification in Java
- String manipulation techniques
- File I/O operations
- Command-line user interface design
- Exception handling
- Logging best practices

## Project Structure
- `CharacterChecker.java` - Main application class with UI and logic
- `InvalidInputException.java` - Custom exception for input validation
